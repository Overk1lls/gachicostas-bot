## Использование

Обратиться к боту в сообщении с помощью тага или по нику:
* Тагнуть бота с вопросом ```кто на сервере ...?```
* Тагнуть бота с вопросом ```... или ...?```
* Тагнуть бота с любым вопросом ```... ?```
* Тагнуть бота с оскорблением ```<tag> ...```
